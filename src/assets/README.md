# Assets Folder

This folder contains static assets for the React application.

## Structure:
- images/ - For image files
- icons/ - For icon files  
- styles/ - For additional CSS files
- fonts/ - For custom font files

## Usage:
Import assets in your components using relative paths:
```javascript
import logoImage from '../assets/images/logo.png';
import './assets/styles/custom.css';
```
